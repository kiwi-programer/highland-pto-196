# highland-pto-196

Static website for Highland Elementary PTO.

## Vercel deployment

This project is ready for Vercel as a static site.

- Root page: `index.html`
- No build command required
- No output directory required

### Deploy

1. Push this repo to GitHub.
2. In Vercel, click **Add New... → Project**.
3. Import this repository.
4. Keep defaults (Framework Preset: **Other**), then click **Deploy**.

### Local preview

You can preview locally with any static server, for example:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.
